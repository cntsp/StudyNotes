### 课程介绍

**课程内容**

- 8周Go语言基础
- 3个实战项目（含1个线上项目）
- 每周作业+课后答疑



**如何学习Go语言？**

- 与其他编程语言对比学习
- 多写多写多写代码
- 在工作中使用Go语言
- 培养自主学习能力
